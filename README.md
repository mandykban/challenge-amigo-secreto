## Descripción del Proyecto

Challenge Amigo Secreto es una aplicación web sencilla que facilita la organización de sorteos de amigo secreto. Permite ingresar una lista de participantes y sortearlos de forma aleatoria, mostrando el resultado en tiempo real. Los participantes sorteados se tachan automáticamente de la lista, permitiendo realizar sorteos consecutivos hasta que no queden participantes o reiniciar el proceso para comenzar de nuevo.

Este proyecto se basa en el desafío ALURA ONE, utilizando el código base HTML y CSS proporcionado. El archivo JavaScript fue desarrollado completamente para personalizar y expandir la funcionalidad de la aplicación.

## Características de la Aplicación

*   ✅  Ingreso de nombres de participantes.
*   ✅  Sorteo aleatorio de un participante a la vez.
*   ✅  Tachado automático del participante sorteado.
*   ✅  Sorteos consecutivos hasta agotar la lista.
*   ✅  Interfaz adaptable para listas largas sin necesidad de desplazamiento.

## Tecnologías Utilizadas

*   **JavaScript:** Desarrollo completo de la lógica de la aplicación.
*   **HTML:** Modificación del código base, incluyendo la adición del botón "Nuevo Sorteo".
*   **CSS:** Ajustes de proporciones para visualizar a los participantes sorteados.

## Modificaciones Realizadas

*   **HTML:** Se añadió el botón "Nuevo Sorteo" para reiniciar el sorteo.
*   **CSS:** Se ajustaron las proporciones para visualizar listas largas sin necesidad de scroll.
*   **JavaScript:** Desarrollo completo de la lógica para gestionar los sorteos y actualizar dinámicamente la lista de participantes.

## Conclusión

Challenge Amigo Secreto simplifica la organización de sorteos, ofreciendo una experiencia digital intuitiva y visualmente atractiva.
